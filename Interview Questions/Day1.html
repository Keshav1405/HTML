Q1. Difference between div and span?

Div stands for "division tag" which is used to group some content together and it's a block level element so it will take the 100% width of the page with a line break before and after it. On the other hand "span tag" is used to write small amount of content and it's an inline element by nature so it will only take as much space it needs and allows other elemets to place themselves beside it.

Q2. Difference between class and IDs?

Class and IDs are both attributes which are given to an HTML element in the opening tag for styling. Class is an attribute which can be given to multiple elements(same class_name) but ID can't be same for more than one element, each element has a unique ID_name.

Q3. What are self closing tags?

Self closing tags are those tags which doesn't have a closing tag.It is also known as empty tags. The attributes for are given in the opening tag itself.

For example:

        br                  ---> Break tag
        hr                  ---> Horizontal rule tag
        img src="#" alt=""  ---> Image tag
        input type="text"   ---> Input tag

Q4. How many types of headings are there ?How to decide which to use where ?

There are 6 types of headings viz a viz h1 , h2 , h3 , h4 , h5 , h6 . They are all used as per the developers discretion with h1 having the biggest font size and highest priority whereas h6 having the least font size among all and lowest priority.

Q5. Difference between block and inline?

Block and Inline are both display properties of an element, every element has a default display property but it can be changed as per the needs of webpage. A "block level" element will take the 100% width of the page with a line break before and after it. Whereas "inline element" will only take as much space it needs and allows other elemets to place themselves beside it.

Q6. Difference between em and strong?

These are the html tags which are used to provide some importance or emphasis on a particular text inside the webpage. "em tag" is used to emphasize text in a way that it should be read in another mood, it makes the text 'italic'. Whereas "strong tag" is used to give importance to a text in the webpage, it makes the text 'bold'.

Q7. What are attributes?

Attributes are used to add more information corresponding to an HTML tag. They are always given in the opening tag in lowercase as they are case sensitive.

For example:

        < a href="#">Content< /a>

Q8. Difference between HTML tags and elements?

HTML tags only comprise of opening tag & closing tag ehereas HTML element consist of opening tag, content & closing tag.

Q9. Explain br tags?

"br tag" is used to give a line break to the HTML webpage. It is a self closing tag.

Q10. What are semantic tags?

Semantic tags are those tags which explain their meaning to both the browser and the developer.

Example of non-semantic tags are div tag and span tag ----> Tells nothing about its content.

Example of semantic tags are Form tag, table tag, header tag, footer tag etc ----> Clearly defines it's content.
